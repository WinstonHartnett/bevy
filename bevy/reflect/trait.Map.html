Expand description

An ordered mapping between Reflect values.

Because the values are reflected, the underlying types of keys and values may differ between entries.

ReflectValue Keys are assumed to return a non-None hash. The ordering of Map entries is not guaranteed to be stable across runs or between instances.

This trait corresponds to types like std::collections::HashMap.

Returns a reference to the value associated with the given key.

If no value is associated with key, returns None.

fn insert_boxed( &mut self, key: Box<dyn Reflect + 'static, Global>, value: Box<dyn Reflect + 'static, Global> ) -> Option<Box<dyn Reflect + 'static, Global>>

Inserts a key-value pair into the map.

If the map did not have this key present, None is returned. If the map did have this key present, the value is updated, and the old value is returned.

fn remove( &mut self, key: &(dyn Reflect + 'static) ) -> Option<Box<dyn Reflect + 'static, Global>>

Removes an entry from the map.

If the map did not have this key present, None is returned. If the map did have this key present, the removed value is returned.
